## Remember!

When you push, check:
```bash
git status              # See what you're about to push
git diff --staged       # Review changes
git add <files>         # Only add source files
git commit -m "..."     # Descriptive message
git push                # Push to your branch
```

Never:
```bash
git push origin main    # ❌ DIRECT TO MAIN - Use PR instead
git add -A              # ❌ Adds everything - Check first!
git push -f             # ❌ Force push (unless recovering from disaster)
```

Good luck! 🎮
